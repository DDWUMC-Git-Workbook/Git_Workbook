## 개념
Working Stage : 현재 작업하고 있는 공간
Staging Area : 변경사항 반영
Local Repository : 작업이 기록되는 곳

.gitignore : 파일이 추적되지 않게 관리해주는 파일
특정 파일, 특정 확장자, 특정 폴더를 추적 해제할 수 있음
(*.txt 혹은 folder_name/)
